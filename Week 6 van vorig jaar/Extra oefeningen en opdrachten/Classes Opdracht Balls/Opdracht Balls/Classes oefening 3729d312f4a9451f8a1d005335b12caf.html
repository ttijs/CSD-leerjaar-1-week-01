Classes oefening

Met deze oefening ga je aan de slag met classes. Classes zijn een manier om je code op te breken in losse objecten, die elk hun eigen logica en gedrag bevatten. Veel talen, zoals C# en javascript, bieden een vorm van classes aan. Volg de stappen en lees de instructies goed door. Kom je er niet uit? Check dan eerst met je leerteam en stap vervolgens naar je docent.

Stap 1:

Maak een nieuwe sketch aan met een canvas van 400 bij 400. Kies een leuke achtergrondkleur. Ik heb (30,40,80) gebruikt.

Stap 2:

Teken een balletje in de draw() functie. Gebruik daarbij de volgende variabelen:

  • let x: 200
  • let y: 200
  • let size: 30

Stap 4:

Maak een nieuw bestand aan in je project genaamd ball.js en voeg deze toe aan je index.html. Doe je dit niet, dan kan sketch.js straks niet bij het ball.js bestand.

Stap 5:

Neem de onderstaande code nauwkeurig over in het ball.js bestand. Uitleg over wat hier staat komt later.

Stap 6:

Verplaats je eigen code van sketch.js naar ball.js:

  • In de update() functie plaats je je eerder geschreven code voor het bewegen.
  • In de display() functie plaats je je code voor het tekenen van de ball.

Om de code weer te laten werken pas je je sketch.js bestand aan:

Uitleg

Constructor:

De constructor is een soort functie die automatisch wordt uitgevoerd bij het aanmaken van een instance van de class. Deze constructor kan ook parameters hebben (hier x, y, size).

Variabelen:

Door ball.speed te gebruiken in sketch.js kom je uit bij de .speed variabele van de ball instance. Iedere ball heeft straks zijn variabelen met een eigen waarde.

Functies:

Een class kan ook functies hebben die je op dezelfde manier aanroept als variabelen.

Stap 7:

Zorg ervoor dat je 50 instances van de ball class hebt die van links naar rechts bewegen. Iedere bal heeft een random x en y waarde.

  1. Maak van de ball variabele in sketch.js een array genaamd balls.
